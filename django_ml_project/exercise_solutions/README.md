# Exercise Solutions (30 marks)

Standalone Python scripts for each exercise part.

## How to Run

### Run All Exercises
```bash
cd django_ml_project
./venv/Scripts/activate
python RUN_ALL_EXERCISES.py
```

### Run Individual Parts
```bash
python exercise_solutions/part_a_rwanda_map.py       # 20 marks
python exercise_solutions/part_b_coefficient_variation.py  # 5 marks
python exercise_solutions/part_c_improve_silhouette.py     # 5 marks
```

## Files

- `part_a_rwanda_map.py` - Rwanda Map Visualization (20 marks)
- `part_b_coefficient_variation.py` - Coefficient of Variation (5 marks)
- `part_c_improve_silhouette.py` - Improve Silhouette Score (5 marks)
